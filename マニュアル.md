## 2. 日常運用

### 自動更新について
毎日13時にMacが自動でCSVを読み込み、GitHubに反映します。
**MacがスリープしていてもOK。ただし電源が切れていると実行されません。**

### 手動で今すぐ更新したい場合
ターミナルを開いて以下を実行：

```bash
bash ~/Documents/eight-men-monthly-ranking/update.sh
```

### 更新ログの確認
```bash
cat ~/Documents/eight-men-monthly-ranking/update.log
```

## 6. ランキングの仕様

### スタッフランキング
- **指標**：指名売上（月次累計）
- **除外**：アシスタント、工藤 邑輔、メモ、フリー
- **兼任スタッフ**：全店舗の売上を合算して表示
- **前日比**：集計中の月は前日との順位変動を表示
- **前月比**：月末最終日のCSVが揃った時点で自動切り替え
- **先月X位**：集計中の月のみ名前の横に表示

### 店舗ランキング
- **指標**：総売上（月次累計）
- **生産性**：総売上 ÷ アクティブスタイリスト数

## 8. トラブルシューティング

### サイトが更新されていない
1. `update.log` を確認してエラーがないか確認
2. Macが13時に起動していたか確認
3. 手動実行で動作確認：`bash ~/Documents/eight-men-monthly-ranking/update.sh`

### pushでエラーが出る（認証エラー）
GitHubのPersonal Access Tokenが期限切れの可能性があります。
1. https://github.com/settings/tokens で新しいTokenを生成（スコープ：`repo`）
2. `update.sh` を手動実行し、Username/Passwordを再入力

### 自動更新を止めたい
```bash
launchctl unload ~/Library/LaunchAgents/com.eightmen.ranking.plist
```

### 自動更新を再開する
```bash
launchctl load ~/Library/LaunchAgents/com.eightmen.ranking.plist
```

### 時刻を変更したい
`~/Library/LaunchAgents/com.eightmen.ranking.plist` の以下を編集：
```xml
<key>Hour</key>
<integer>13</integer>   ← 変更したい時間（24時間表記）
<key>Minute</key>
<integer>0</integer>    ← 分
```
編集後：
```bash
launchctl unload ~/Library/LaunchAgents/com.eightmen.ranking.plist
launchctl load ~/Library/LaunchAgents/com.eightmen.ranking.plist
```
